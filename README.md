# blog

https://shimajiteppei.github.io/

## Prerequisite

- Docker
- Node.js

## Tasks

### Install, Update

```sh
npm install
```

### Develop

Start auto-reload server.

```sh
npm start
```

Stop server.

```sh
npm stop
```

Restart server.

```sh
npm restart
```

### Lint

Proofread markdowns expect for drafts.

```sh
npm test
```

### Build & Preview

```sh
npm run generate
```
